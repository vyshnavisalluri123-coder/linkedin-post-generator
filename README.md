# LinkedIn Post Generator 💼

A Streamlit app that generates LinkedIn posts with customizable tone, length, bullet points, and hashtags.

## Features
- Multiple tones: Professional, Casual, Story, Bold
- Short or long post length
- Bullet point formatting
- Auto-generated hashtags
- Export post as text file

## Installation
```bash
pip install -r requirements.txt
streamlit run vyshapp.py
